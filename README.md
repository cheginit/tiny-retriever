## Installation

Choose your preferred installation method:

### Using `pip`

```console
pip install tiny_retriever
```

### Using `micromamba` (recommended)

```console
micromamba install -c conda-forge tiny_retriever
```

Alternatively, you can use `conda` or `mamba`.
